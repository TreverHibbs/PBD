# Build Instructions

## All
1. Have git and build essentials installed
2. Either install raylib as described here https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux
or use CMAKE and have it download the raylib dependency for you with fetch content.

## Ubuntu
1. Use the existing makefile or the makefile generated with CMAKE to build the binary.
'cmake -B ./build' should generate the makefile.
